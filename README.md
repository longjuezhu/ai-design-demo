# AI Design Demo

这是一个使用 AI 辅助完成的 UI/UX 原型练习项目。

## 项目背景
本项目模拟一个移动端电商 AI 商品推荐页面，目标是帮助目标不明确的用户快速理解推荐理由并完成商品浏览。

## 使用工具
- Figma / MasterGo：界面设计与原型
- ChatGPT / Claude：需求拆解与文案优化
- Cursor / v0：辅助生成可交互 Web Demo
- GitHub：项目管理与展示

## 我的角色
- 需求拆解
- 信息架构设计
- UI 页面设计
- AI 辅助原型验证
- 交互状态整理

## 项目亮点
- 使用 AI 辅助进行竞品分析和用户需求拆解
- 将设计规范转化为组件和页面结构
- 补充 loading / empty / error 等真实产品状态
- 输出可访问的 Web Demo
